# 🚗 Vehicle Data Explorer

**Interactive Used Vehicle Market Analytics Dashboard**

[Live Demo on Render](https://aplicacionweb2.onrender.com/)

---

## 📌 Project Overview

Vehicle Data Explorer is an interactive web application built with **Streamlit** for exploring and analyzing used vehicle listings in the United States. The dashboard allows users to investigate pricing patterns, mileage distribution, and manufacturer-level trends through dynamic visualizations and filters.

This project demonstrates skills in **data analysis, interactive dashboard development, and cloud deployment**.

---

## 🎯 Project Objective

* Explore the used vehicle market through interactive analytics
* Analyze relationships between **price** and **mileage**
* Compare vehicle trends across different manufacturers
* Build a user-friendly dashboard for data-driven decision-making

---

## 🚀 Key Features

### 📊 Interactive Visualizations

* **Mileage Distribution Histogram:** Explore how vehicle mileage is distributed across listings
* **Price vs Mileage Scatter Plot:** Identify trends, outliers, and pricing behavior
* **Average Price by Manufacturer:** Compare market positioning across brands

### ⚡ Dynamic Filtering

* Filter data by **manufacturer**
* Adjust the **price range** interactively
* Generate updated insights in real time

### 📈 Dashboard Metrics

* Total number of vehicles listed
* Average vehicle price
* Average vehicle mileage

---

## 🎯 Business Impact

This dashboard helps users quickly identify pricing patterns, evaluate vehicle value, and make more informed decisions in the used vehicle market through data-driven insights.

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Kcruz4/VehicleDataExplorer.git
cd VehicleDataExplorer
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the Streamlit app:

```bash
streamlit run app.py
```

4. Open the local URL generated by Streamlit in your browser.
